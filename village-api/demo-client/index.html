How to Use the Demo

Follow these steps to get autocomplete working end-to-end.

1

Start the server already done if you see this

From the project root, run both the API and frontend together:

cd village-api npm run dev

This starts:
• API server at http://localhost:3000
• Frontend at http://localhost:5173

2

Register a business account

Go to http://localhost:5173/register and fill in the form.
Use a non-gmail/yahoo email (e.g. test@mycompany.com). Your account will be pending approval initially.

3

Approve the account as admin

Log in as admin at http://localhost:5173/login:

Email: admin@villageapi.com Password: Admin@123456

Then go to Users → click the green tick next to your new account to approve it.
Also click into the user detail and grant state access — tick the states you want to search, then click Update Access.

4

Generate an API key

Log out of admin, then log in as your business account at http://localhost:5173/login.
Go to API Keys → enter a name → click Generate Key.
Copy the ak_... key that appears. The secret is shown only once — you don't need it for the demo.

5

Paste the key into the Demo tab

Switch to the Demo tab above. In the config bar at the top:
• Leave API URL as http://localhost:3000/api/v1
• Paste your ak_... key into API Key

Then type any village name in the autocomplete field — results appear instantly.

6

Import data if autocomplete returns no results optional

If the DB is empty, run the importer against your Excel sheets:

cd village-api node scripts/import-data.js --dir ../Excel_Sheets

This populates all states, districts, sub-districts and villages. Depending on file count it may take a few minutes.

7

Open the standalone HTML demo optional

The file demo-client/index.html is a self-contained demo you can open directly in a browser — no React needed. Just open it with:

xdg-open demo-client/index.html # Linux open demo-client/index.html # macOS

Edit the API_KEY variable at the top of the script block to your key, and make sure the API server is running on port 3000.
